# 🌍⚽ La Porra del Mundial 2026 – Reglament Oficial

Lloc web oficial del reglament de **La Porra del Mundial 2026**, creat pel **Grup Olga**.

## 🔗 Visitar el Lloc Web

> Publica'l a GitHub Pages per obtenir un domini gratuït!

## 📋 Contingut del Reglament

El lloc web inclou tots els apartats del reglament:

1. **Inscripció i Fons Econòmic** — Terminis i constitució del premi
2. **Estructura de Premis** — Distribució del 1r, 2n i 3r premi (50% / 25% / 15%)
3. **Fases del Joc** — Grups, vuitens, quarts, semifinals i final
4. **Sistema de Puntuació** — Punts per fase i type d'encert
5. **Hores Límit de la Final** — Terminis específics del tram final
6. **Criteris de Desempat** — 5 criteris ordenats per prioritat
7. **Regles de Validesa** — Condicions de quiniela vàlida/invàlida

## 🚀 Com publicar-lo GRATIS (GitHub Pages)

### Pas 1 – Crea un compte a GitHub
Ves a [github.com](https://github.com) i crea un compte gratuït.

### Pas 2 – Crea un nou repositori
- Prem el botó `+` → `New repository`
- Nom suggerit: `porra-mundial-2026`
- Marca-lo com a **Public**
- Prem `Create repository`

### Pas 3 – Puja els fitxers
Des de la pàgina del repositori:
- Prem `uploading an existing file`
- Arrossega els 3 fitxers: `index.html`, `styles.css`, `main.js`
- Prem `Commit changes`

### Pas 4 – Activa GitHub Pages
- Ves a `Settings` → `Pages` (menú lateral esquerre)
- A **Source**, selecciona `Deploy from a branch`
- Branca: `main`, carpeta: `/ (root)`
- Prem `Save`
- Espera ~2 minuts i el teu lloc estarà disponible a:
  `https://[el-teu-usuari].github.io/porra-mundial-2026/`
